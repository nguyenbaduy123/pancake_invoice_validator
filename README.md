# InvoiceCake

Elixir library for validating e-invoice data with country-specific rules.

## Installation

Add `pancake_invoice_validator` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:pancake_invoice_validator, git: "git@github.com:pancake-vn/pancake_invoice_validator.git"}
  ]
end
```
